TP - Fonctions Asynchrones en JavaScript

Instructions

Dans ce TP, vous allez manipuler des fonctions asynchrones en JavaScript. La fonction fetchSimulatedData() est déjà fournie et simule une requête à un serveur avec un délai aléatoire et la possibilité d'échec.

  1. Créez une fonction asynchrone permettant de gérer l'appel à fetchSimulatedData()
  2. Ajoutez un event handler au bouton qui appellera votre fonction asynchrone
  3. Affichez les données récupérées dans la zone prévue
  4. Implémentez un compteur pour les erreurs rencontrées
